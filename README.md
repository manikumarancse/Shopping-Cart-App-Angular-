# ShoppingCart

This project was developed in Angular with version 12.2.18.

## Note

When you first clone this project, you need to run `npm install` in the terminal. It installs dependency packages for this Angular project.
Then run `ng serve` in the terminal to run your project

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
